Конечные точки

  • GET /api/v1/passwords/{api_key}

    Возвращает все пароли, связанные с пользователем, используя его API-ключ.

    • Ответ: JSON-объект, содержащий пароли пользователя.
    • Ошибка: 404 - Пользователь не найден
    • Ошибка: 500 - Ошибка доступа к базе данных

  • GET /api/v1/username/{api_key}

    Возвращает имя пользователя, связанное с пользователем, используя его API-ключ.

    • Ответ: JSON-объект, содержащий имя пользователя.
    • Ошибка: 404 - Пользователь не найден
    • Ошибка: 500 - Ошибка доступа к базе данных

  • POST /api/v1/passwords/add/{api_key}

    Добавляет новый пароль для пользователя, используя его API-ключ.

    • Запрос: JSON-объект, содержащий имя сайта и пароль.
    • Ответ: JSON-объект с сообщением об успехе.
    • Ошибка: 404 - Пользователь не найден
    • Ошибка: 500 - Ошибка доступа к базе данных

Аутентификация

Аутентификация API осуществляется с использованием API-ключей. Каждому пользователю назначается уникальный API-ключ, который используется для аутентификации запросов.

Обработка ошибок

Ошибки возвращаются с использованием кодов состояния HTTP и JSON-объекта, содержащего сообщение об ошибке.
